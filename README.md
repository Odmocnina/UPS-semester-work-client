Klient app for players in Rock-paper-scissors-lizard-Spock game. Klient can connect, if two players are available the game is started. Players can choose their moves and the winner is determined based on the rules of the game. The game is played in rounds, and the first player to reach a certain number of wins is declared the overall winner.

Done as semester work in KIV/UPS (Introduction to Computer Networks) (2024/2025) at ZCU/FAV.
